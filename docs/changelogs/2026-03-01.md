# Daily Change Log: March 1, 2026

## New Features

**Curated Verified Foods** (#429)
~100 common foods (banana, chicken breast, olive oil...) now ship with USDA-verified macros, green shield badge, and smart serving defaults. Search "banana" and get 118g (medium) as the default — not some mystery 100g blob. Verified foods win dedup over community data but yield to your custom QuickFoods.

**Free-Text Recipe Creation** (#426)
Type "2 eggs, toast with butter, 2 strips bacon" → AI parses it → DB enriches the macros → review modal opens pre-populated. Also ships 11 backlog fixes: shared `parseAIJson` utility, unified `RecipeIngredient` type, prompt injection hardening, rate limit differentiation, and more (#415–#425).

## Bug Fixes

- Changing portion size in food edit now **recalculates macros proportionally** instead of leaving stale values (#405)
- Input length guards on chat history (20 msgs / 2000 chars) and food search (200 chars) (#404)

- Mobile workout chat fully overlays sticky headers, proper safe-area padding (#401)
- Food search results survive the cancel round-trip — no more re-searching (#401)
- Macro displays rounded to whole numbers across 6 files (#401)
